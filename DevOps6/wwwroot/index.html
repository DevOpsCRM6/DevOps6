MSIS 4363 – Advanced Web Application Development

Mark Babcock, Caleb Jackson, Rylee Davis

DevOps and Career Readiness

This course covers the essential tools and practices used by modern DevOps engineers:

  • Azure Cloud Services – Building and deploying scalable applications.
  • CI/CD Pipelines – Automating builds, testing, and deployments using GitHub Actions and Azure DevOps.
  • Containerization – Packaging applications with Docker for consistency across environments.
  • Source Control – Managing repositories, branches, and merges using Git and GitHub.
  • Agile Project Management – Integrating GitHub with Jira for continuous collaboration and delivery.

Together, these skills prepare students for entry-level DevOps engineering internships and full-time positions, emphasizing automation, collaboration, and deployment efficiency.
